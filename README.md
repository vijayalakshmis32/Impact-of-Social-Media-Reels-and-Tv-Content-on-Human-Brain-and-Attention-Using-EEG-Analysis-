##  Overview

This project investigates the effects of different media consumption patterns on human brain activity using Electroencephalography (EEG). It compares cognitive responses while participants are exposed to short-form social media content, television programs, educational videos, and silent resting conditions.

The implementation simulates realistic EEG recordings, extracts frequency-band features, computes cognitive biomarkers, performs statistical analysis, and generates publication-quality visualizations to better understand attention, mental fatigue, and cognitive engagement.

---

##  Objectives

- Record and analyze EEG signals under four media conditions.
- Compare cognitive responses across **5, 15, and 30-minute** exposure durations.
- Analyze EEG frequency bands (Delta, Theta, Alpha, Beta, Gamma).
- Quantify attention variation and cognitive engagement.
- Measure mental fatigue using EEG biomarkers.
- Estimate dopamine-like stimulation associated with short-form social media.
- Generate comprehensive statistical and graphical reports.

## Key Findings

- Short-form social media produces the highest attention-switching frequency.
- Educational videos maintain the most stable attention and cognitive engagement.
- Television content results in moderate cognitive engagement with slower fatigue accumulation.
- Silent resting produces dominant Alpha activity associated with relaxation.
- Extended exposure to social media leads to increased mental fatigue and cognitive overload.
- Daily social media usage negatively correlates with sustained attention.
